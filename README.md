# AnimeWorld-PHP-Kodik-API-
AnimeWorld – это полнофункциональный веб-сайт для просмотра аниме, построенный на PHP и использующий API плеера Kodik. Проект включает в себя регистрацию пользователей, систему закладок, историю просмотров, комментарии, глобальный чат, а также адаптивный дизайн для мобильных устройств. Код полностью сгенерирован с помощью нейросети deepseek.
