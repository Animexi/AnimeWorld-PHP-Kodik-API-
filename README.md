# Animesite-PHP-Kodik-API
AnimeWorld – это полнофункциональный веб-сайт для просмотра аниме, построенный на PHP и использующий API плеера Kodik. Проект включает в себя регистрацию пользователей, систему закладок, историю просмотров, комментарии, глобальный чат, а также адаптивный дизайн для мобильных устройств. Код полностью сгенерирован с помощью нейросети deepseek и предназначен для ознакомления, обучения и доработки.

## ✨ Основные возможности

- **Пользовательская система** – регистрация, авторизация, редактирование профиля (аватар, никнейм, bio).
- **Закладки** – добавление аниме в список с категориями: «В планах», «Смотрю», «Просмотрено», «Любимое».
- **История просмотров** – автоматическое сохранение просмотренных серий с указанием времени.
- **Отметки просмотренных серий** – визуальный индикатор (кнопка серии становится зелёной).
- **Интеграция с Kodik API** – получение информации об аниме, сезонах, сериях, плеере; данные кэшируются в SQLite и файлах.
- **Поиск** – по названию, с фильтрацией по жанрам и сортировкой (по рейтингу, дате добавления, названию). Пагинация с подгрузкой через AJAX.
- **Страница аниме** – подробная информация: описание, жанры, студия, рейтинг, список серий (разбит по сезонам).
- **Плеер** – встроенный iframe-плеер Kodik, навигация между сериями (кнопки «←» и «→»).
- **Комментарии** – древовидные комментарии с лайками/дизлайками, возможностью редактирования и удаления (только свои). Комментарии привязаны к конкретному аниме и отображаются на странице просмотра.
- **Адаптивный дизайн** – корректное отображение на мобильных устройствах (меню сворачивается, сетка подстраивается).
- **Кастомный контекстное меню** – правый клик открывает меню с копированием ссылки и открытием в новой вкладке.
- **Кастомные модальные окна** – подтверждение удаления комментария и других действий.

## 🛠 Технологии

- **Backend:** PHP 7.4+ (без фреймворков)
- **База данных:** SQLite (PDO)
- **API:** Kodik API (требуется токен; в коде уже используется публичный токен `8ebf915587af48d001d33127ae55dcb3`)
- **Frontend:** HTML5, CSS3 (адаптивная вёрстка), JavaScript (нативный, без jQuery)
- **Кэширование:** файловая система + SQLite для хранения данных об аниме

## 🚀 Установка и запуск

1. **Склонируйте репозиторий** или скачайте архив.
2. Поместите файлы в корневую директорию вашего веб-сервера (например, `htdocs` для XAMPP).
3. Убедитесь, что у веб-сервера есть права на запись в папку проекта (для создания кэша и базы данных):
   - Папка `cache/` (создаётся автоматически)
   - Файл `anime111.db` (создаётся автоматически)
4. Настройте веб-сервер так, чтобы точка входа была `index.php`.
5. Откройте сайт в браузере по адресу `http://localhost/`.

**Важно:** Проект использует публичный токен Kodik API. Если он перестанет работать, замените его на свой в файле `index.php` (константа `KODIK_TOKEN`). Получить токен можно на [официальном сайте Kodik](https://kodikapi.com/) (требуется регистрация).

## 📖 Использование

- **Главная страница** – отображает популярные и недавно добавленные аниме.
- **Поиск** – введите название или выберите жанр. Результаты можно сортировать и подгружать по кнопке «Загрузить ещё».
- **Аниме** – нажмите на карточку, чтобы перейти на страницу с описанием и списком серий. Нажмите на номер серии для просмотра.
- **Просмотр** – на странице плеера доступны кнопки переключения на предыдущую/следующую серию, а также блок комментариев внизу.
- **Регистрация/Вход** – необходимы для создания закладок, истории, комментариев и чата.
- **Профиль** – после входа вы можете редактировать свой профиль (никнейм, bio, аватар). Видна статистика.
- **Закладки** – добавляйте аниме в разные списки, управляйте категориями.
- **История** – список просмотренных серий.
